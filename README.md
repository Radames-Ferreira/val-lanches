# Val Lanches

Site institucional desenvolvido para o trailer de lanches **Val Lanches**, localizado em frente ao SENAC.

**Projeto acadêmico** da disciplina de **Programação Web** - SENAC.

---

### ✨ O que o site tem

- Design moderno e responsivo
- Cardápio de lanches
- Informações de localização e contato
- Seção "Sobre nós"
- Fotos do trailer

---

### 🛠 Tecnologias usadas

- HTML5

- JavaScript
- Bootstrap
